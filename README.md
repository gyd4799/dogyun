## 狗云有哪些比较实在的地方

**按时计费可随时暂停**：弹性云不用的时候直接暂停，只扣硬盘和 IPv4 的费用，CPU/内存停着不计钱，比很多家强。

**自助换IP**：控制台里自己操作就能换 IPv4 和 IPv6，不用发工单等人工。

**可退款**：充值成功一个月内，只要余额还在，可以原路退回充值款项。相比那些"充多少花多少不退"的商家，这一点还是比较良心的。

**机龄计划是长期用户的隐形福利**：越用越久，免费流量额度越涨，适合打算长期稳定用一台服务器的朋友，时间越长性价比越高。

**99.9%+ 在线率承诺**：官方表示非人为不可抗力情况下，在线率接近 99.98%。
